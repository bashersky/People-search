Cоздан cайт для поиска людей в ВКонтакте.
Получилось реализовать такие функции как:
Вывод по 10 человек с последующей подгрузкой.
Поиск людей через поисковую строку.
При нажатии на пользователя загружается 2 страница с информацией о пользователе.

Для роботы сайта необходимо получить токен по ссылке
https://oauth.vk.com/authorize?client_id=6834608&display=page&redirect_uri=&scope=&response_type=token&v=5.92
